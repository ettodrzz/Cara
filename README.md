# Presentación

Creé este repositorio con el propósito de ser una guía de cómo **instalar**, **configurar** o **personalizar** las herramientas que habitualmente uso como estudiante de desarrollo web.

Especialmente con Linux, no soy un experto, pero sé lo intimidante que puede llegar a ser. Por eso la finalidad de esta especie de bitácora, ayudar a otros coders novatos (como yo) a entender algunas de las tecnologías del mundo *techie* o *geek*.

Espero que esto sea un recurso de apoyo para ser más productivo y seguir aprendiendo.
